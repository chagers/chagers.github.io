The Enumerable Method: cycle

Say that again?

05.01.15

Today I'll be talking about the Enumerable method called cycle. Ruby Docs provides the following documentation on this method:

Pretty dry explanation, right? Let's see if we can make it a bit more understandable and applicable. Global Nerdy gives a great and simple definition for the cycle method:

Given a collection, [Enumerable#cycle] creates an infinitely-repeating ordered source of its items.

So, now that we get what the cycle method will do. Why would we want to use it? Well, say you have a list of household chores that you want to assign to all your household members. Ok, let's say you live in a huge house with a lot of roommates and you just don't want to manually write this list out. You also want to give everyone a day of the week to complete their chore so they stay on top of it. Enumerable#cycle can help you!
